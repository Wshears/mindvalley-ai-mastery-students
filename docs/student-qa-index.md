## Pro Tip: Learning to Prompt

Every Q&A answer includes a **Claude Code Prompt** you can copy-paste. These prompts teach you HOW to ask Claude Code for help effectively.

**Pattern to learn:**
1. Give context (what you're doing)
2. State the problem clearly
3. Request specific help
4. Mention your skill level

Example:
```
I'm trying to [task] for the MindValley AI Mastery course.

I'm getting [error/issue].

Please help me:
1. [First thing]
2. [Second thing]
3. [Third thing]

I'm new to this, so please explain each step.
```
